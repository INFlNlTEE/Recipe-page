# 🍳 Simple Omelette Recipe Page

A clean and responsive recipe page built with semantic HTML and modern CSS. This project focuses on proper structure, accessibility, and visually appealing UI design.

---

## 📌 Overview

This is a frontend project that displays a simple omelette recipe, including:

* Preparation time
* Ingredients list
* Step-by-step instructions
* Nutritional information

The goal of this project is to practice:

* Semantic HTML
* Clean CSS styling
* Responsive design
* UI structure and layout

---

https://inflnltee.github.io/Recipe-page/

---

## 🛠️ Built With

* HTML5 (Semantic structure)
* CSS3 (Custom properties, layout, styling)

---

## 🎨 Design Features

* Responsive layout (mobile-first approach)
* Clean card-based UI
* Consistent typography using:

  * **Young Serif** (headings)
  * **Outfit** (body text)
* Custom color palette using CSS variables
* Styled sections for:

  * Preparation time highlight
  * Ingredients & instructions
  * Nutrition table
